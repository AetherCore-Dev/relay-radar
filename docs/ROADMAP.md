## 🔲 Phase 2: 排名网站 (下一步)

**目标**: relayradar.com — 中转站排名网站，用户只看不传

**架构**:
```
用户 → relayradar.com (香港服务器) → 只看排名
                  ↑
后台测试服务器 → 自己购买+测试各家中转站 → 结果存数据库
```

| 任务 | 预计时间 |
|------|----------|
| Next.js静态站 + 排名页面 | 3天 |
| 中转站详情页 (五维雷达图 + 趋势) | 2天 |
| 在线成本计算器 | 1天 |
| 免责声明 + 关于页面 | 1天 |
| 部署到香港服务器 (阿里云) | 1天 |
| 手动测试5-10家中转站 | 2天 |
| 发掘金文章 | 1天 |

**排名维度**:
- ⚡ 延迟 (20%) — TTFT + P50/P95
- 🛡️ 稳定性 (20%) — 7天错误率
- 💰 价格 (20%) — 官方定价对比
- 🔬 真实性 (25%) — 模型验真分数
- 👥 口碑 (15%) — 用户评分

**数据来源**: 全部由我们自己测试，不需要用户上传任何东西

## 技术栈

- **核心引擎**: Node.js ESM, 零依赖
- **CLI**: Node.js, 零依赖
- **网站**: Next.js 15 + React 19
- **部署**: 阿里云香港 + Cloudflare CDN
- **CI/CD**: GitHub Actions
- **测试**: Node.js内置test runner
