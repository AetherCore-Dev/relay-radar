## ✅ Phase 2: 排名网站（已完成 2026-04-02）

**目标**: relayradar.com — 中转站排名网站，用户只看不传

| 任务 | 状态 | 说明 |
|------|------|------|
| Next.js静态站 | ✅ | Next.js 16 + React 19，output: 'export' |
| 首页排名表格 | ✅ | 五维评分条 + 评级标签 + 价格倍率 + 标签 |
| 详情页 | ✅ | 统计卡片 + 五维条形图 + SVG趋势图 + 定价表 + 警告 |
| 工具页 | ✅ | 交互式成本计算器，实时对比直连 vs 中转站 |
| 关于页 | ✅ | 测试方法论 + 评分体系 + 隐私声明 + 免责声明 |
| 模拟数据 | ✅ | 5个中转站（推荐/可用/可疑/造假），7天历史 |
| 暗色主题 | ✅ | CSS Variables，响应式，零外部UI依赖 |
| 构建验证 | ✅ | 5个页面全部静态生成，本地预览通过 |

**数据架构**:
```
用户 → relayradar.com (GitHub Pages / 香港服务器) → 只看排名
                  ↑
后台 → 我们购买各家中转站 → 跑 relay-radar rank → 结果更新到 data/rankings.json
```

**排名维度**:
| 维度 | 权重 | 数据来源 |
|------|------|----------|
| ⚡ 延迟 | 20% | 自动化探测（TTFT + P50/P95） |
| 🛡️ 稳定性 | 20% | 连续7天监测（错误率 + 可用时间） |
| 💰 价格 | 20% | 人工收集 + 计费审计 |
| 🔬 真实性 | 25% | 8维指纹验真（最高权重） |
| 👥 口碑 | 15% | 未来开放用户评分 |

## 技术栈

| 层 | 技术 | 理由 |
|----|------|------|
| 核心引擎 | Node.js ESM, **零依赖** | 最小信任面，无供应链风险 |
| CLI | Node.js 内置 `parseArgs` | 不需要commander/yargs |
| 测试 | Node.js 内置 `node:test` | 不需要jest/vitest |
| 网站 | Next.js 16 + React 19 | 静态导出，全球CDN |
| 部署 | GitHub Pages → 阿里云HK | 免费起步 → 按需升级 |
| CI/CD | GitHub Actions | push自动测试+部署 |
| 域名 | Cloudflare Registrar | 便宜 + DNS管理 + CDN |
